# Task 1: Check if a Number is Even or Odd

## Description:
This Python script allows the user to input an integer and then determines whether the number is **even** or **odd** using an `if-else` statement.

## Output Example:
Enter a number: 7
7 is an odd number.

Enter a number: 12
